# Some dots here and there

Hyprland setup heavily based on [HyDE](https://github.com/HyDE-Project/HyDE) and [Omarchy](https://github.com/basecamp/omarchy) using OpenCode to do the dirty work of moving and coping stuff.

## Dependecies

### Packages
```sh
$ yay -S base-devel nvim git waybar dunst walker swww hyprlock wlogout kitty zsh starship fastfetch rofi zip unzip lazyvim nwg-displays impala bluetui wiremix
$ flatpak install [discord, stremio, spotify, telegram, obsidian]
```

```sh
$ yay -S base-devel nvim git waybar dunst walker awww hyprlock wlogout kitty zsh dolphin
$ flatpak install [discord, stremio, spotify, telegram, obsidian]
```
